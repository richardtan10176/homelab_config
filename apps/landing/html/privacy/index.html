Privacy Policy

Last updated: 19 July 2026

Summary

This is a personal website consisting of static pages. It does not collect personal information, set cookies, run analytics, or ask you to submit anything.

What this site does not do

  • No cookies or similar storage are set.
  • No analytics, tracking pixels, or advertising services are used.
  • No JavaScript runs on these pages.
  • No third-party resources are embedded — no external fonts, scripts, or images. Loading a page requests files from this site only.
  • There are no forms, accounts, or newsletters.

Data that is unavoidably processed

Serving a web page requires a network connection, and connections leave records. Specifically:

  • Cloudflare. Traffic to this domain is proxied by Cloudflare, which terminates TLS and processes connection data such as your IP address, the requested URL, and your user agent. This processing is governed by Cloudflare's own privacy policy, not this one.
  • Server logs. The origin web server writes standard access logs containing the requesting IP address, timestamp, requested path, response status, and user agent. These are operational records used for debugging and keeping the site running.

This information is not used to build profiles, is not combined with any other data, and is not sold, rented, or shared with third parties. It is not used for marketing.

Links to other sites

This site links to external sites, including projects on subdomains and third-party profiles. Those destinations have their own privacy practices, and this policy does not cover them.

Changes

If the site changes in a way that affects this policy — for example if analytics or a contact form were added — this page will be updated and the date above revised.

Contact

Questions about this policy can be sent to you@example.com.
